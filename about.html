About the Creators

Thank you for checking out My Christian Chat Server!

       This web application was originally created as a group project for a class at Liberty University that grew into the current product that you see today. This application strives to be a Christian-friendly chat that any person can use without the fear of seeing any unsavory media that you may see on other websites.

Its friendly fun for all ages, so check out the chat room and stay a while!
